Starter project for a simple baroque concert & events site inspired by Viertelbarock.

## Features added in this branch
- Header, hero, events list, footer components
- Responsive layout and Google fonts (Playfair Display + Inter)

## Run

Install dependencies:

```powershell
npm install
```

Dev server:

```powershell
npm run dev
```

Open dev URL in browser:

```powershell
Start-Process "http://localhost:5173"
```
